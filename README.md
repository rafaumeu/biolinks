## Overview

A modern **link-in-bio** platform built with **Laravel 12**, featuring customizable pages, analytics tracking, and SEO optimization. Uses Pest for testing and PHPStan for static analysis.

## Features

- Customizable link pages with themes
- Click analytics and visitor tracking
- SEO-optimized URLs and meta tags
- Admin dashboard for link management
- Responsive design with TailwindCSS + DaisyUI
- Full test coverage with Pest

## Tech Stack

| Technology | Purpose |
|---|---|
| **Laravel 12** | PHP framework |
| **TailwindCSS** | Utility-first CSS |
| **DaisyUI** | Component library |
| **Pest** | Testing framework |
| **PHPStan** | Static analysis |

## Getting Started

```bash
git clone https://github.com/rafaumeu/biolinks.git
cd biolinks
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
